J NOTE · GitHub Pages 설정 가이드

아래 순서대로 진행하면 어디서든 URL로 J NOTE에 접속할 수 있습니다.

1
GitHub 레포지토리 만들기
직접 해야 함
  1. github.com 접속 → 로그인
  2. 오른쪽 상단 + 버튼 → New repository
  3. Repository name: jnote
  4. Public 선택 (GitHub Pages 무료 사용 조건)
  5. Create repository 클릭
2
파일 업로드
직접 해야 함
  1. 생성된 레포지토리 페이지에서 uploading an existing file 클릭
  2. 아래 폴더의 파일 2개를 드래그해서 업로드:
    C:\Users\1\Desktop\jnote-github\
📄 index.html — J NOTE 본체
📄 .nojekyll — GitHub Pages 오류 방지 파일
  1. 하단 Commit changes 클릭
💡 .nojekyll 파일이 안 보이면 파일 탐색기 상단 → 보기 → 숨김 항목 체크
3
GitHub Pages 활성화
직접 해야 함
  1. 레포지토리 상단 Settings 탭 클릭
  2. 왼쪽 메뉴 Pages 클릭
  3. Branch: main 선택 → 폴더: / (root) → Save
  4. 1~3분 기다리면 URL 생성됨
https://[내 GitHub 아이디].github.io/jnote/
이 URL로 PC, 노트북, 스마트폰 어디서든 접속 가능
⚠️ 구글 캘린더 연동은 localhost에서만 작동 → URL 접속 시 캘린더 연동 불가 (나머지 기능 모두 정상)
4
이후 업데이트 방법
✓ 자동
  1. J NOTE 기능이 업데이트되면 새 index.html 파일을 받게 됩니다
  2. GitHub 레포지토리 → index.html 클릭 → 연필 아이콘(Edit) → 내용 붙여넣기 → Commit
  3. 또는 파일을 다시 드래그 업로드해도 됩니다
💡 데이터(매매일지·일기·가계부)는 각 기기 브라우저에 별도 저장됩니다.
기기 간 동기화: 설정 → 데이터 내보내기 → 다른 기기에서 데이터 가져오기
