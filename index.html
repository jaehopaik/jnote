Google Calendar 양방향 동기화
⚠️ Google Calendar는 localhost에서 실행해야 합니다.
바탕화면의 start_jnote.bat을 실행 후 http://localhost:8080/J%20NOTE.html로 접속하세요.

Google Cloud Console 설정:
1. console.cloud.google.com → 새 프로젝트 생성
2. Google Calendar API 활성화
3. OAuth 2.0 클라이언트 ID 생성 (웹 애플리케이션)
4. 승인된 JavaScript 원본에 http://localhost:8080 추가
5. 클라이언트 ID를 아래에 입력
데이터 백업 / 복원
다른 기기(노트북 등)에서 사용하려면 데이터를 내보낸 후 JSON 파일을 옮겨 가져오기 하세요.
